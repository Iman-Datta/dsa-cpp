# C++ Data Structures and Algorithms

This repository contains my practice implementations of **Data Structures and Algorithms using C++**.
The goal of this repository is to improve **problem-solving skills, algorithmic thinking, and coding efficiency**.

## Repository Structure

The repository is organized topic-wise for easier learning and revision.

```
00_Algorithm        -> Core algorithm implementations
00_NPTEL_course     -> Problems and implementations from NPTEL course
01_Basic            -> Basic C++ and foundational programs
02_ArrayBasic       -> Basic array operations and problems
03_VectorBasic      -> STL vector usage and related problems
04_STL              -> C++ Standard Template Library examples
05_Basic_math       -> Mathematical algorithms and problems
06_Basic_recursion  -> Recursion fundamentals and examples
07_Basic_Hashing    -> Hashing concepts and problems
09_ArrayProblem     -> Array-based coding problems
```
